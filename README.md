# China Toilet Finder

Find public toilets across China with walking routes and 6-language support. Built with Amap (高德地图).

No build tools needed — just open `index.html` in your browser, or serve with any static server.

## Features

- Search nearby public toilets within 1500m
- Walking route navigation with distance & time
- Deep links to Apple Maps & Google Maps
- 6 languages: English, 中文, 한국어, 日本語, Español, Français
- PWA installable on mobile home screen
